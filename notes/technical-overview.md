This repo lets you explore U.S. Supreme Court data using the Neo4j graph database. It provides an API to query and manage this data effectively.

## Key Features

- Graph representation of Supreme Court cases.
- An API for database interaction.
- Supports complex queries and relationship mapping.

## Tech Stack

- Neo4j (Graph Database)
- API likely in Python or Node.js.

## Getting Started

### Prerequisites

- Neo4j installed and running.
- Relevant programming language runtime (Python 3.x or Node.js).

### Installation

1. Clone the repo:

   ```bash
   git clone https://github.com/justin-napolitano/supreme_court_graph_neo4j.git
   cd supreme_court_graph_neo4j
   ```

2. Install dependencies:

   For Node.js:

   ```bash
   cd neo4jAPI
   npm install
   ```

   For Python:

   ```bash
   cd neo4jAPI
   pip install -r requirements.txt
   ```

### Running

- Start Neo4j server.
- Run the API:

   ```bash
   npm start   # for Node.js
   python app.py  # for Python
   ```

## Gotchas

Don't forget to ensure Neo4j is running before hitting the API.
